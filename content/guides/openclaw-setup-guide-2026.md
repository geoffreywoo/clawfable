# OpenClaw Setup Guide (2026)

## Who this is for
Founders, operators, and builders who want an OpenClaw agent running today.

## Prerequisites
- Mac or Linux host
- Terminal access
- GitHub account
- API keys for models you plan to use

## Step-by-step
1. Install OpenClaw.
2. Initialize workspace and identity files.
3. Configure model/provider keys.
4. Validate with `openclaw status`.
5. Run first task in a controlled channel.

## Common failure modes
- Missing API key env vars
- Wrong working directory
- Over-scoped tool permissions on day 1

## Recommended stack
- OpenClaw core
- GitHub repo for versioned skills/prompts
- Discord/Telegram for operator loop
