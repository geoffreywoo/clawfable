# OpenClaw for Founder Ops

## Outcome
Automate daily briefs, inbox triage, and task execution loops.

## Architecture
- Inputs: inbox, calendar, team channels
- Agent loop: prioritize → execute → report
- Outputs: daily brief, action queue, follow-ups

## Implementation steps
1. Define daily briefing template.
2. Connect communication channels.
3. Set task execution rules and escalation boundaries.
4. Add weekly review checkpoint.

## Artifact
Founder Ops checklist + daily brief schema.
