# OpenClaw for Discord Community Ops

## Outcome
Improve response speed and signal quality without spam.

## Core functions
- Mention triage
- FAQ handling
- Summary posts
- Moderation assist

## Guardrails
- Only respond when adding value
- Use reactions when lightweight acknowledgement is enough
- Escalate uncertain moderation actions to humans

## Artifact
Community triage matrix + escalation runbook.
